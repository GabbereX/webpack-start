# Стартовая WebPack сборка
____
### Запустить сервер
>npm start
### Сформировать проект для разработчика
>npm run build:dev
### Сформировать проект для продукта
>npm run build:prod
___
## В сборку входят пакеты:
- dependencies
  - react
  - react-dom
  - react-router-dom
- devDependencies
  - autoprefixer
  - clean-webpack-plugin
  - cross-env
  - css-loader
  - dotenv-webpack
  - editorconfig
  - favicons-webpack-plugin
  - html-loader
  - html-webpack-plugin
  - mini-css-extract-plugin
  - postcss-loader
  - prettier
  - sass
  - sass-loader
  - style-loader
  - svg-sprite-loader
  - ts-loader
  - tsc
  - typescript
  - webpack
  - webpack-cli
  - webpack-dev-server
